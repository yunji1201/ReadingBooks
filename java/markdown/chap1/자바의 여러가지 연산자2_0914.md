# 0914 [자바의 여러가지 연산자2]

8. 복합 대입 연산자
    - 대입 연산자와 다른 연산자를 함께 사용함
        - += , -= , *= , /= , %=

    - 비트 연산자
        - 비트 연산자는 정수에만 사용할 수 있다
            - ~, <<, >>, &, |,
            - <<=, >>=, >>>=, &=, |=, ^=

    - 조건 연산자
        - 삼항 연산자
        - 조건 식의 결과가 true, false인 경우에 따라 다른 식이나 결과가 수행됨
        - 제어문 중 조건문을 간단히 표현 가능
        - 조건식 ? 결과1 : 결과2
        - ex: int num = (5>3) ? 10 : 20;)
